Кусары Услуги — локальный запуск

1) Распакуй папку.
2) Запусти start_windows.bat
3) Открой в браузере: http://127.0.0.1:5000

Важно:
- Смотреть объявления можно без регистрации.
- Чтобы разместить объявление / поставить оценку / написать комментарий — нужно войти.
- Аккаунты + объявления + комментарии/оценки сохраняются в app.db (SQLite) рядом с server.py.

Админ-панель:
- Открывай в браузере: http://127.0.0.1:5000/admin
- Вход — через обычный логин/пароль, но доступ есть только у пользователей с ролью admin.

Как создать первого админа (самый простой способ):
1) Открой start_windows.bat
2) Раскомментируй и задай:
   set KUSARY_ADMIN_IDENTIFIER=admin
   set KUSARY_ADMIN_PASSWORD=strongpass
3) Перезапусти start_windows.bat


Проверка что всё работает:
- Открой: http://127.0.0.1:5000/api/health
  Должно быть: {"ok": true, "db_ok": true, ...}

Если "не грузит" или "ничего не работает":
1) Убедись, что ты НЕ открываешь index.html двойным кликом (file://). Нужно запускать start_windows.bat и открывать http://127.0.0.1:5000
2) Распакуй проект в папку, где есть права на запись (например Desktop/Документы). В папках типа Program Files SQLite может не создать app.db.
3) Если порт 5000 занят — открой start_windows.bat и добавь:
   set "KUSARY_PORT=5050"
   Потом открывай: http://127.0.0.1:5050
4) Если окно сервера закрывается — в конце start_windows.bat есть пауза. Скопируй ошибку и отправь мне.
