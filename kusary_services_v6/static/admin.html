Кусары Услуги ADMIN
—

Вход администратора

Используй логин/e-mail и пароль администратора.

Как создать/назначить админа
  1. Открой файл start_windows.bat и добавь перед запуском две переменные окружения:
set KUSARY_ADMIN_IDENTIFIER=admin
set KUSARY_ADMIN_PASSWORD=strongpass

После перезапуска сервера пользователь admin будет создан (или станет админом).
